## 📸 How to Select What to Capture

When you click "Start Recording", you'll see three options:

### Option 1: Application Window (Recommended)
```
✅ Best for: Single application like QuickBooks
✅ Pros: Focused, keeps other apps private
✅ Example: "QuickBooks Desktop 2024"
```

### Option 2: Entire Screen
```
✅ Best for: Multi-app workflows
✅ Pros: Captures everything, natural workflow
⚠️ Note: Shows all windows and notifications
```

### Option 3: Browser Tab
```
✅ Best for: Web apps only
✅ Pros: Clean capture of web applications
❌ Won't work for: Desktop software like QuickBooks
```

## 🔒 Privacy Considerations

### What Others WON'T See:
- ❌ Other applications (when using "Application Window" mode)
- ❌ Other browser tabs (when using "Browser Tab" mode)
- ❌ Notifications from other apps
- ❌ Anything minimized or hidden

### What Others WILL See:
- ✅ The application/screen you selected
- ✅ Anything you type or click
- ✅ Pop-ups or dialogs that appear
- ✅ Content visible in the selected window

### Best Practice:
🔐 Use "Application Window" mode for sensitive work - it only captures the specific app you select!

## 🎯 Capture Tips

### For Desktop Applications (like QuickBooks):
✅ **DO**: Use "Application Window" mode
✅ **DO**: Maximize the window for better screenshots
✅ **DO**: Close unnecessary windows
❌ **DON'T**: Minimize the application during capture

### For Web Applications:
✅ **DO**: Use "Browser Tab" for web-only workflows
✅ **DO**: Zoom to comfortable reading size (Ctrl/Cmd + Plus)
✅ **DO**: Hide bookmarks bar for cleaner screenshots

### For Multi-Application Workflows:
✅ **DO**: Use "Entire Screen"
✅ **DO**: Arrange windows before starting
✅ **DO**: Close distracting applications
✅ **DO**: Silence notifications

## 🆘 Troubleshooting

**"I don't see my application in the list"**
- Make sure the application is open (not minimized)
- Try selecting "Entire Screen" instead
- Restart your browser

**"Screen sharing keeps stopping"**
- Don't minimize the captured window
- Don't switch to "Entire Screen" if capturing an app
- Check that the application allows screen capture

**"Capture is black/blank"**
- Some applications block screen capture (security feature)
- Try "Entire Screen" mode
- Update your graphics drivers
- Try a different browser
