# DocuCapture - Smart Process Documentation Tool

Transform the way you document processes with screen capture, step-by-step guides, and intelligent optimization.

## 🚀 Quick Start

### Getting Started
1. Click on the DocuCapture link provided by your administrator
2. Choose your tool:
   - **Capture Tool** - Record and document processes
   - **Process Analyst** - Download and run locally for analysis

### That's it!
DocuCapture works entirely in your browser - no installation needed.

## 📝 How to Use

### For Remote Users (Capturing Processes)

1. **Visit the Capture Tool**
   - Go to your deployed website
   - Or open `capture.html` in any browser

2. **Record Your Process**
   - Click **"Start Recording"**
   - Select the screen/window to capture
   - Perform your task step-by-step
   - Click **"Capture Step"** after each action
   - Add descriptions for each step

3. **Export for Analysis**
   - Click **"🤖 Export for Process Analysis"**
   - Save the `.json` file
   - Send it to yourself or the analyst

### For Analysts (Process Optimization)

1. **Open the Analysis Tool**
   - Download `analyze.html` to your computer
   - Double-click to open in your browser
   - (This keeps your API access secure and private)

2. **Import Process**
   - Click **"📥 Import Process"**
   - Select the `.json` file you received
   - All steps will load automatically

3. **Analyze with Analysis**
   - Click **"🤖 Analyze with Analysis"**
   - Wait for the system to analyze the workflow
   - Review efficiency score and recommendations

4. **Get Insights**
   - See efficiency score (0-100)
   - Review key inefficiencies
   - Get optimization suggestions
   - Identify automation opportunities
   - View step-by-step AI notes

5. **Export Results**
   - Export as HTML (with Analysis insights)
   - Export as Markdown
   - Print or save as PDF

## 💡 Features

### Capture Tool (Remote)
- ✅ Screen recording with live capture (captures ANY application - desktop software, web apps, or entire screen)
- ✅ Works with desktop applications (QuickBooks, Excel, Photoshop, SAP, etc.)
- ✅ **Intelligent sensitive data detection** - Automatically identifies passwords, credit cards, SSNs, API keys
- ✅ **Multi-method redaction** - Blur, pixelate, or black box sensitive information
- ✅ **Visual security alerts** - Clear warnings when sensitive data is detected
- ✅ Screenshot at each step
- ✅ Manual image upload
- ✅ Rich text descriptions
- ✅ Drag-to-reorder steps
- ✅ Export to multiple formats
- ✅ Export for Process analysis

### Analysis Tool (Local)
- 🤖 Intelligent process optimization
- 📊 Efficiency scoring (0-100)
- 💡 Smart recommendations
- 🔍 Inefficiency detection
- ⚡ Automation opportunities
- 📝 Step-by-step Analysis insights
- 📤 Enhanced exports with Analysis data

## 🔒 Privacy & Security

- **Capture Tool**: No data is sent to any server (works entirely in browser)
- **Sensitive Data Detection**: Automatically alerts you to potential passwords, credit cards, SSNs, and other sensitive information
- **Redaction Tools**: Blur, pixelate, or black box sensitive data before sharing (see [REDACTION_GUIDE.md](REDACTION_GUIDE.md))
- **Analysis Tool**: Only sends process descriptions to Analysis Engine (no screenshots unless you choose to export them)
- **API Keys**: Never exposed in the remote tool
- **Your Data**: Stays on your device until you explicitly export

## 🤝 Support

Having issues? 

1. **Check browser compatibility** - Use latest Chrome, Firefox, or Edge
2. **Enable screen sharing** - Some browsers require HTTPS for screen capture
3. **Clear browser cache** - If you see old versions after updates
4. **Check file format** - Export files should be `.json` format
