🚀 Quick Start Guide

Get started with DocuCapture in 2 minutes!

← Back to Home

⚡ For End Users

1️⃣ Access DocuCapture

  • Open the link provided by your system administrator
  • Or open index.html from your local files
  • Bookmark it for easy access

2️⃣ Start Creating Documentation

Using the Capture Tool:

  1. Click "Start Recording"
  2. Select what to share (choose your application window)
  3. Grant permission when prompted
  4. Perform your task step-by-step
  5. Click "Capture Step" after each action
  6. Add descriptions for what you did
  7. Export when finished

3️⃣ Learn More

  • Click "Interactive Tutorial" for a guided walkthrough
  • Check the User Guide for detailed instructions
  • Submit Enhancement Requests to suggest improvements

🎯 Common Tasks

Create Training Documentation

  1. Open Capture Tool
  2. Record the process
  3. Add clear descriptions
  4. Redact any sensitive data
  5. Export as HTML or PDF
  6. Share with trainees

Get Process Optimization

  1. Capture your process
  2. Export for Analysis (JSON format)
  3. Open Process Analyst tool
  4. Import the JSON file
  5. Click "Analyze Process"
  6. Review efficiency score and recommendations

Redact Sensitive Information

  1. After capturing, look for red alert banners
  2. Click "Redact Data" on any step
  3. Choose redaction style (Blur, Pixelate, or Black Box)
  4. Click and drag over sensitive areas
  5. Apply and save

⚙️ No Installation Required!

Everything runs in your web browser:

  • No software to install
  • No plugins needed

  • Works on Windows, Mac, Linux

🆘 Troubleshooting

Screen capture not working?

  • Use Chrome 72+, Edge 79+, Firefox 66+, or Safari 13+
  • Make sure you grant screen recording permission
  • Try selecting "Application Window" instead of "Entire Screen"

Redaction not saving?

  • Make sure you click "Apply & Save"
  • Check that browser allows localStorage
  • Try refreshing the page

Can't export documentation?

  • Check browser allows file downloads
  • Try a different export format
  • Make sure you have at least one step captured

Process Analyst not working?

  • Make sure analyze.html is running locally on your computer
  • Verify you imported a valid JSON file
  • Check that you have an internet connection (for analysis features)
