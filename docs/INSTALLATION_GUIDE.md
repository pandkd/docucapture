## Global Keyboard Shortcuts

These work **system-wide** - even when DocuCapture is minimized!

| Shortcut | What It Does | When to Use |
|----------|-------------|-------------|
| `Ctrl + Shift + R` | Start/Stop Recording | Begin or end documentation session |
| `Ctrl + Shift + S` | Capture Step | **Most important!** Capture current screen |
| `Ctrl + Shift + D` | Show/Hide Window | Quickly access DocuCapture |
| `Ctrl + Shift + T` | Toggle Timeline View | Switch between List and Timeline |
| `Ctrl + Shift + G` | Smart Group Steps | Automatically organize steps |
| `Ctrl + Shift + H` | Show Shortcuts Help | See all keyboard shortcuts |
| `Ctrl + Shift + E` | Go to Export Menu | Jump to export options |
| `Ctrl + Shift + A` | Export for Analysis | Quick export as JSON |

## Troubleshooting

### Installation Issues

**Problem:** "Windows protected your PC" won't let me install

**Solution:**
1. Click "More info"
2. Click "Run anyway"
3. This is safe - it's just a new app without a paid certificate

---

**Problem:** Installation fails or crashes

**Solution:**
1. Right-click installer
2. Select "Run as administrator"
3. Try again

---

**Problem:** "App can't run on your PC"

**Solution:**
- DocuCapture requires 64-bit Windows
- Check your Windows version:
  - Press `Win + Pause/Break`
  - Look for "System type"
  - Must say "64-bit operating system"

---

### First-Run Issues

**Problem:** App won't start after installation

**Solution:**
1. Restart your computer
2. Try launching again
3. If still failing:
   - Right-click Desktop shortcut
   - Select "Run as administrator"

---

**Problem:** No system tray icon appears

**Solution:**
1. Click up arrow in system tray (^)
2. DocuCapture icon might be hidden
3. Drag it to main tray area

---

### Recording Issues

**Problem:** Can't select screen to capture

**Solution:**
- Close other screen recording software
- Restart DocuCapture
- Try "Run as administrator"

---

**Problem:** Floating button doesn't appear

**Solution:**
- Check if recording actually started
- Look in bottom-right corner
- Try minimizing other windows

---

### Performance Issues

**Problem:** App runs slowly

**Solution:**
- Close other applications
- You need 1 GB free RAM
- Check Task Manager (`Ctrl + Shift + Esc`)

---

**Problem:** Screenshots take long to capture

**Solution:**
- Use lower resolution screen
- Close memory-intensive apps
- Restart DocuCapture

## Tips for Success

### Best Practices

1. **Start Small**
   - Document a simple 5-step process first
   - Get comfortable with the controls
   - Then tackle complex workflows

2. **Use Keyboard Shortcuts**
   - Much faster than clicking
   - Works from any app
   - Learn `Ctrl + Shift + S` first

3. **Keep Recording Sessions Short**
   - 10-20 steps is ideal
   - Break long processes into parts
   - Easier to manage and edit

4. **Add Descriptions Immediately**
   - While the context is fresh
   - Click each step after capturing
   - Type a brief description

5. **Use Smart Grouping**
   - After capturing 10+ steps
   - Press `Ctrl + Shift + G`
   - Auto-organizes into sections
